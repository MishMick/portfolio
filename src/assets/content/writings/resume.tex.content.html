About

Attributes

  • Developed the résumé using LaTeX.
  • Résumés for both roles can be generated using the same codebase.
  • Uses minimal dependencies, nothing outside of TeX Live.
  • Produces beautifully typeset outputs in two themes.
  • Clean, well-documented, easy-to-fork and easy-to-modify code.
